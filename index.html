Projects

Mr. Whiskers' Blog

A personal blog layout built using semantic HTML. This project is a blog-style webpage that showcases structured content centered around a cat named Mr. Whiskers. Built entirely in pure HTML, the page includes multiple sections such as an engaging welcome message, an “About” section, a series of blog posts with clear titles and content, and a contact block with phone and email info. The purpose of this project was to demonstrate proper HTML document structure, effective use of semantic tags, and thoughtful content organization — laying a strong foundation for future styling with CSS or frameworks like Tailwind.

Event Hub

Event Hub is a semantic HTML project that showcases a structured webpage for managing and displaying events. It includes clearly separated sections for Upcoming Events and Past Events, each using semantic tags such as

  • section
  • article
  • nav
  • header
to ensure accessibility and logical document flow. The project demonstrates a strong understanding of semantic HTML and how to organize content in a maintainable and readable structure — laying the foundation for future enhancements with CSS, interactivity, or integration with real data sources.
